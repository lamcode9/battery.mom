# Neon Database Setup (Recommended)

## Step 1: Create Neon Account (2 minutes)

1. Go to **https://neon.tech**
2. Click **"Sign Up"** (use GitHub login)
3. Click **"Create a project"**
4. **Project name**: `evcompare-sea` (or any name)
5. **Region**: Choose closest to you (e.g., `Singapore` or `Tokyo`)
6. **PostgreSQL version**: 16 (default is fine)
7. Click **"Create project"**

## Step 2: Get Connection String (1 minute)

1. After project creation, you'll see the **Connection string**
2. It looks like: `postgres://username:password@ep-xxx-xxx.region.aws.neon.tech/neondb?sslmode=require`
3. **Copy the connection string** (click "Copy" button)

## Step 3: Create .env File

Once you have the connection string, I'll create the `.env` file for you!
